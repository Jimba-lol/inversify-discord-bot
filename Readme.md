# My Discord Bot

### Running the Discord Bot / Express Application:
###### to compile in watch mode:
`npm run watch`
###### to compile:
`npm run build`
###### to test:
`npm run test`
###### to run:
`npm run start`
### Running the Angular Application:
###### to test:
`ng test`
###### to run (compiles automatically):
`ng serve`
