# My Discord Bot

### Running the Discord Bot:
###### to compile in watch mode:
`npm run watch`
###### to compile:
`npm run build`
###### to test:
`npm run test`
###### to run:
`npm run start`
###### to register commands:
`npm run register-commands`

### Setup
put custom audio files here:\
`src/resource/sound`
